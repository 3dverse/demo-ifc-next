# demo-ifc-next


Demo IFC and Digital Twin web application powered by 3dverse.

### 3dverse Console setup
- Create a 3dverse account at https://console.3dverse.com/
- Upload your IFC file

### Preprocess IFC data
```
cd public/scripts
python create_json.py
python post_process.py
```

### Getting started

```
npm install
npm run dev
```
